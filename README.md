# Weather Journal App

## Description

Get temperature from an API using Zip and country code.

## tools and languages used to create the project

1. HTML5
2. CSS3
3. Javascript
4. Node.JS
5. Express.JS


## To Start

1. git clone https://github.com/taha-elmasir/Weather-Journal-App.git
2. cd MyReads-App
3. npm install
4. npm start
